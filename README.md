## Project Description
This project focuses on reducing hallucinations in Large Language Models (LLMs) using prompt validation and response scoring techniques. Different prompt engineering strategies such as closed-ended prompting, contextual framing, and fact-based prompting were evaluated to analyze their effectiveness in reducing hallucinated responses.

The project includes dataset preparation, response evaluation, hallucination analysis, and a Streamlit-based implementation for testing and comparison.
